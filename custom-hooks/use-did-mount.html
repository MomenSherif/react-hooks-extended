useDidMount

About​

Calls a callback on mount.

Import​

import { useDidMount } from 'react-hooks-extended';

Baisc example​

  • JavaScript
  • TypeScript
useDidMount(() => {
  console.log('useDidMount');
});
useDidMount(() => {
  console.log('useDidMount');
