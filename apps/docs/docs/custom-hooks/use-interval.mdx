## About

Repeatedly calls a function or executes a code snippet, with a fixed time delay between each call.

## Import

```javascript
import { useInterval } from 'react-hooks-extended';
```

## Baisc example

<Tabs groupId="lng">
  <TabItem value="javascript" label="JavaScript">

```jsx
function Example() {
  const [count, setCount] = useState(0);

  useInterval(() => setCount(c => c + 1), 1000);

  return (
    <div>
      <p>{count}</p>
    </div>
  );
}
```

## Arguments

| Argument          | Type                 | Description                                                                                                                  | Required | Default value |
| :---------------- | :------------------- | :--------------------------------------------------------------------------------------------------------------------------- | :------- | :------------ |
| `callback`        | `Function`           | A function to be executed every delay milliseconds. The first execution happens after delay milliseconds.                    | true     |               |
| `delay`           | `number`             | The time, in milliseconds (thousandths of a second), the timer should delay in between executions of the specified function. | false    | `0`           |
| `options`         | `UseIntervalOptions` | Interval options                                                                                                             | false    |               |
| `options.enabled` | `boolean`            | Enable or disable the interval                                                                                               | false    | `true`        |
