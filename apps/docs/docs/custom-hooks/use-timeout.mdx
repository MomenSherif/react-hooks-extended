## About

Sets a timer which executes a function the timer expires.

## Import

```javascript
import { useTimeout } from 'react-hooks-extended';
```

## Arguments

| Argument          | Type                | Description                                                                   | Required | Default value |
| :---------------- | :------------------ | :---------------------------------------------------------------------------- | :------- | :------------ |
| `callback`        | `Function`          | A function to be executed after the timer expires                             | true     |               |
| `delay`           | `number` or `Date`  | The time in 'milliseconds' or 'Date' in the future that the timer should wait | false    | `0`           |
| `options`         | `UseTimeoutOptions` | Timer options                                                                 | false    |               |
| `options.enabled` | `boolean`           | Enable or disable the timeout                                                 | false    | `true`        |

## Playground

```jsx live
function Playground() {
  const [isEnabled, toggleIsEnabled] = useToggle();
  const [state, setState] = useState('');

  useTimeout(
    () => {
      setState('useTimeout');
    },
    1000,
    { enabled: isEnabled }
  );

  return (
    <div>
      <h1>{state}</h1>
      <button onClick={toggleIsEnabled}>
        {isEnabled ? 'Disable' : 'Enable'}
      </button>
    </div>
  );
}
```
