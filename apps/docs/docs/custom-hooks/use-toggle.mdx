## About

Toggle between `booleans` or `custom data` easily with useToggle.

## Import

```javascript
import { useToggle } from 'react-hooks-extended';
```

## Arguments

| Argument         | Type                  | Description                 | Required | Default value |
| :--------------- | :-------------------- | :-------------------------- | :------- | :------------ |
| `initialState`   | `T`                   | Initial state value         | false    | `false`       |
| `toggleFunction` | `(prevState: T) => T` | Custom toggle functionality | false    | `s => !s`     |

## Return

Array with a stateful value, a function to toggle it and a function to force update the state to certain

| Return value | Type                 | Description                  |
| :----------- | :------------------- | :--------------------------- |
| `state`      | `T`                  | Current state                |
| `toggle`     | `Function`           | Toggle between values        |
| `setState`   | `(state: T) => void` | Set state to a certain value |
