## About

Calls a callback after each update in the dependency list.

## Import

```javascript
import { useDidUpdate } from 'react-hooks-extended';
```

## Baisc example

<Tabs groupId="lng">
  <TabItem value="javascript" label="JavaScript">

```jsx
function Example() {
  const [count, setCount] = useState(0);

  useDidUpdate(() => {
    console.log(`Updated count ${count}`);
  }, [count]);

  return (
    <div>
      <p>{count}</p>
      <button onClick={() => setCount(c => c + 1)}>Increment</button>
    </div>
  );
}
```

## Arguments

| Argument   | Type             | Description                                                             | Required | Default value |
| :--------- | :--------------- | :---------------------------------------------------------------------- | :------- | :------------ |
| `callback` | `EffectCallback` | Imperative function that can return a cleanup function.                 | true     |               |
| `deps`     | `DependencyList` | If present, effect will only activate if the values in the list change. | false    | `undefined`   |
